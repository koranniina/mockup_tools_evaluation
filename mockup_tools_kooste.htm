Concept.ly

How it works

Create new project, add name and select project type (iPhone-Portrait, iPad-Portrait, Desktop, iPhone-Landscape, iPad-Landscape). Drag and drop or browse your files. With Concept.ly you can only upload screens. Theres no master templates, widgets, fonts and cant create any interactions. Save your project as PDF or ZIP-file.

 

Concept.ly in pictures


Easel

 

How it works

Create new document, add name and select template (Blank canvas, Marketing place, Blog, Home page). With Easel you can create your mockup with lot of different elements and with your own images. Interactions cant be done. Save your project as HTML or PNG files.

 

Easel in pictures

 

 

InVision

 

How it works

Create new project, add name and select project type (Desktop, portrait and landscape iPhone, portrait and landscape Android, portrait and landscape iPad, portrait and landscape Android tablet). Drag and drop or browse your files. With InVision you can only upload screens. Theres no master templates, widgets, fonts and cant create any interactions. Save your project as PDF-file.

 

 

InVision in pictures

 

 

 

Justinmind

 

How it works

Download software. Create new prototype and select a device (Web, iPhone 5, iPad, Android, Android tablet, Custom, iPhone 4, Google Glass). If you select empty prototype you can edit its width and height. You can also select existing prototype file (.vp), images or example prototype (Creativio, Responsive, Adaptive, Health Assurance, Online Store). With Justinmind you can create your mockup with lot of different elements or with your own images and add lot of interactions. Save your project as DOC, HTML or ODF files.

 

Justinmind in pictures

 

 

Proto.io

 

How it works

Create new project, add title, select project template (Blank template, Tab menu app for iPhone 5/5S, Tab menu app for Android, Sliding menus app for iPhone 5/5S, Sliding menus app for Android, Social app for iPhone 5/5S, Social app for iPad 2/Mini, Social app for Android) and select a device (iPhone 5/5S, iPhone 4/4S, iPhone 3/3S, iPad 1/2, iPad Retina, Nexus One, Google Nexus 7, Google Nexus 5, Samsung Galaxy S5, Samsung Galaxy Tab 4, Android Wear, HTC Windows, Lumia 1020, Ativ S Windows, Microsoft Surface Tablet, Website, Custom device). With Proto.io you can create your mockup with lot of different elements or with your own images and add lot of interactions. Save your project as PDF, HTML5 or ZIP-files.

ProtoShare

 

How it works

Add account domain name (name will be part of the url address) and add name to your project, you will automatically start working with blank template. With ProtoShare you can create your mockup with lot of different drop components, assets, masters and templates or with your own images and add lot of interactions. Save your project as DOC or HTML -files.

 

ProtoShare in pictures

 

 

UXPin

 

How it works

Create new project, add name and select project type (Wireframe, iPhone app, Responsive website). With UXPin you can create your mockup with lot of different drop components, assets, masters and templates or with your own images and add lot of interactions. Save your project as PDF, PNG or HTML -files.
